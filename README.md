# CPFHub.io OpenAPI Specification

This repository contains the official OpenAPI Specification for the CPFHub.io API.

## What is CPFHub.io?

CPFHub.io is a REST API that provides comprehensive CPF consultation services in Brazil. It returns essential identity information such as full name, gender, and date of birth for any Brazilian CPF number, with high accuracy, low latency, and full LGPD compliance.

## Why use this OpenAPI Specification?

This OpenAPI specification serves as the single source of truth for the CPFHub.io API, enabling:

*   **Automated Integration**: Facilitates seamless integration with various tools, platforms, and AI agents that support OpenAPI.
*   **Code Generation**: Allows for automatic generation of client SDKs, server stubs, and documentation in multiple programming languages.
*   **AI Agent Discoverability**: Provides a machine-readable description of the API's capabilities, allowing AI agents and Large Language Models (LLMs) to understand and interact with the CPFHub.io API effectively.
*   **Consistency**: Ensures a consistent and up-to-date API contract across all SDKs and integrations.
